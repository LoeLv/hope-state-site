# 诸神圣榜美化执行计划

## 目标

将排行榜重构为「复古神性档案室」：前三名是冠冕陈列，后续名次是分阶卷宗，命途、神祇和分数均可一眼辨认，同时保持现有检索、筛选、公开卷宗入口与移动端功能。

## 已实施

1. 档案材质分层
   - 标题区采用黄铜细边、暗纹与缓慢流光。
   - 前三名采用黑曜石冠冕卡，榜首有独立微光。
   - 观测台、筛选区、势力图和命途回廊采用羊皮与黄铜分层。

2. 数据层级
   - 总分为第一视觉层级，登神分为第二层级，觐见分为弱化层级。
   - #4 至 #12 使用高阶金边，#13 至 #74 为标准卷宗行，#75 以后使用弱化边框。
   - 每行有命途专属侧条，区分生命、沉沦、文明、混沌、存在、虚无。

3. 可视化与联动
   - 神祇势力图以热度柱显示各神祇在册人数。
   - 圣榜观测台增加最盛命途占比环图。
   - 命途回廊展示人数、冠首与头像，点击可筛选对应命途；悬停可预览高亮榜单行。

4. 交互与响应式
   - 排序切换、筛选、检索、榜单行、冠冕卡均保留可访问的按钮交互。
   - 高登神分条目保留低频金色微光。
   - 窄屏下冠冕、观测台、势力图和命途回廊自动改为单列或多行网格。

## 验收

- 桌面及 390px 窄屏已完成渲染截图检查。
- 命途回廊筛选已验证，只保留对应命途的榜单条目。
- `node --check app.js` 与 `git diff --check` 通过。
