# 希望之州

诸神愚戏主题的群内排行榜与个人职业面板网站。

当前版本包含：

- 公开排行榜：所有人可看，只展示昵称、职业、神明、命途和分数。
- 公开职业战斗面板：点击排行榜某人，只能看基础信息、职业特性、血量、攻击和基础职业规则，不暴露天赋。
- 私密个人面板：输入昵称 + 本人暗语后，才能看到自己的天赋和私密备注，并可自助修改暗语、公开短记、私密备注和天赋。
- 管理录入：管理员统一录入玩家信息、暗语、天赋和分数，职业信息由职业资料库自动匹配。
- 批量导入：支持从 Excel/在线表单整理为 TSV 后粘贴；必须包含“昵称、暗语、职业、登神分、觐见分、公开短记、私密备注、天赋”8 列，分数列为空或不是非负整数时会拒绝导入。
- 批量分数结算：管理员可按“信仰 职业 昵称 +登神分 +觐见分”逐行录入，登神范围 -20 到 20，觐见范围 -3 到 3。
- 图片导出：公开面板和私密面板都可以导出图片。
- Supabase 在线模式：见 `supabase/DEPLOY.md`。

默认不填 Supabase 配置时，网站使用浏览器 `localStorage` 演示数据。填好 `app.js` 里的 `config` 后，网站会通过 Supabase Edge Function 读取在线数据。

## 打开方式

直接双击 `index.html` 即可打开。

如果要用本地服务预览：

```powershell
python -m http.server 8812
```

然后访问：

```text
http://127.0.0.1:8812
```

## Supabase

部署步骤在：

```text
supabase/DEPLOY.md
```

关键隐私规则：

- 暗语不放在前端。
- 前端只拿公开排行榜字段。
- 天赋只由 Edge Function 在暗语验证通过后返回。
- 玩家自助修改必须先通过昵称 + 原暗语验证，不能修改职业、登神分或觐见分。
- 管理录入和打分需要 `ADMIN_KEY`。

SQL 执行顺序：

1. `supabase/schema.sql`
2. `supabase/hope_professions_seed.sql`
