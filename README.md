# 希望之州

诸神愚戏主题的新简版网站。

当前第一版保留：

- 个人界面：登记、编辑、删除州民档案。
- 排行榜：按总评、登神分、觐见分排序。
- 打分系统：给已登记成员增减登神分、增加觐见分，并记录结算日志。

当前不包含旧站的副本投稿、评论、天赋、匹配、微信小程序和 Supabase 后端。数据先保存在浏览器 `localStorage`，可以通过页面里的“导出数据 / 导入数据”备份和迁移。

## 打开方式

直接双击 `index.html` 即可打开。

如果要用本地服务预览：

```powershell
python -m http.server 8812
```

然后访问：

```text
http://127.0.0.1:8812
```
